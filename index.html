Quick Protocol Reference

Data packets are sent as UTF-8 strings terminated by a newline character (\n):

D-PAD Codes

Sends on press: UP, DOWN, LEFT, RIGHT.
Sends on release: up, down, left, right.

Joystick & Tilt Codes

Sends: X+xx,Y+yy\n (where xx, yy are signed integers -90 to 90, e.g. X+00,Y-45).

Action buttons & Sliders

Buttons transmit: Press A, B, C, D; Release a, b, c, d.
Sliders transmit: cVal\n (Slider 1) or xVal\n (Slider 2) padded to 3 digits (e.g. c090, x180).

Setup & Programming Guide

Follow these steps to connect and control your micro:bit robot using STEM Smart Labs Controller:

  1. Ensure your micro:bit is powered on and Bluetooth is enabled on your computer or phone.
  2. Flash your micro:bit with a program that listens for UART serial data over Bluetooth. See the templates below.
  3. Click the Pair Micro:bit button in the top bar to scan and connect!

MakeCode (Blocks) Programming

In Microsoft MakeCode, configure the following block setup:

  1. Open MakeCode, click Extensions, and search/add bluetooth.
  2. In on start, insert the bluetooth uart service block.
  3. Create a on bluetooth connected block to display a checkbox.
  4. Create a on bluetooth disconnected block to display a cross.
  5. Create a on rx line received event block. In it, read the line value, clean it, and write conditional statements (e.g. if line == "UP" then driveForward() and if line == "up" then stop()).
